Projects and Publications

Technical Projects


Seagate Technologies - Alteryx Language Processing Macro
During my internship at Seagate Technologies, I was tasked with using Python to create a Natural-Language-Processing application for the purpose of analyzing a variety relevent text data. By employing various linguistic principles as well as various natural language processing techniques in addition to analysis tools such as SciPy and Pandas, the application which I developed provided topic and sentiment modeling capabilities to a customer-focused team of analysts. By deploying the macro, analysts and researchers are able to derive important quantitative data from vast arrays of qualitative inputs. With additional focus on ease-of-use and a priority placed on user-experience, the tool which I developed can be easily deployed within a standard Alteryx workflow for simple additions and alterations.

In addition to the technical responsibilities involved with development, I was able to lead a team of analysts in performing user-acceptance-testing prior to deployment. This multi-step process involved proofing the application for technical vulnerabilities and subjective UX flaws. In addition to user acceptance testing, I drafted and published relevant documentation for the tool for ease of use and a comprehensive description of its capabilities. This project was a major success and continues to be leveraged to create a more responsive customer experience today.


Restaurant Data Tableau Dashboard

Through the construction of relevant series IDs, this dashboard will continue to be updated to illustrate pertinent trends for Colorado restaurants.


Campus Odyssey - Scavenger Hunt Website
Campus Odyssey is a web application developed to serve as a hub for creating, customizing, and mediating scavenger hunts. Through its different capabilities, users are able to create scavenger hunts filled with customizable tasks and post them for others to access. Users are able to upload photos of their completing of tasks for points, which are visible on the scavenger hunt's leaderboard as well as the site's image feed.

The site is built on an architecture consisting of standard EJS views, bootstrap customization, JavaScript-based functionality and a Node.Js integration with a NoSQL server. The application also accesses third-party functionality for image uploads and docker composing.

This project remains private for academic integrity reasons, but I would be happy to discuss its functionality, architecture, and development process. Additionally, if you have any scavenger hunt needs, I am your guy.


Hash Table Dictionary
My Digital Dictionary is a C++ command-line application which leverages chained-hashing and search trees to create an efficient method for parsing the english language. By either appending all words to a single chained hash table or reserving memory by building letter-specific tables, users are able to search for any definition in seconds (frequently in a few thousandths of a second).

Please view the project's GitHub repository (linked above) for more relevant documentation and a more in-depth look into the structures which make this application possible.

The game orients around a point system, where players gather points by visiting cities, winning battles, and acquiring armies and heroes through in-game currency. Reducing Student Inequities (RSI) - Transfer Qualitative Study
Through my involvement in Leeds Student Government, the Reduced Student Inequities pillar group conducted a qualitative survey with the aim of identifying factors which drive a student's decision to transfer to the University of Colorado, with the ultimate goal of increasing transfer rates to CU in the interest of improving the student body.
